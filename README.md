alfdaumsearch
=============

 자동완성 가능한 Daum 키워드 검색과 Daum 사전 검색을 가능하도록 하는 Alfred Workflow

 
 
사용법
----

 * `ds ...` : 일반 키워드 검색
 * `dk ...` : 국어사전 검색
 * `dh ...` : 한자사전 검색
 * `de ...` : 영어사전 검색
 * `dj ...` : 일어사전 검색
 * `dc ...` : 한자사전 검색

적용되어 있는 것 이외에 기능 추가(베트남어 사전 등)를 원하시면 요청바랍니다.


외부 모듈 사용
-----------
Alfred-workflow(https://github.com/deanishe/alfred-workflow)라는 모듈을 사용하였습니다.
